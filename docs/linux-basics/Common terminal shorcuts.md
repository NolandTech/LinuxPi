Here’s a list of useful keyboard shortcuts for navigating and editing in the terminal:

Move to the beginning of the line: Ctrl + A

Move to the end of the line: Ctrl + E

Delete from cursor to the end of the line: Ctrl + K

Delete from the beginning of the line to the cursor: Ctrl + U

Clear the entire line: Ctrl + A (move to start) then Ctrl + K

Move one word forward: Ctrl + Right Arrow

Delete one word backward: Ctrl + W

Delete one word forward: Ctrl + Delete

Move one character backward: Ctrl + B

Move one character forward: Ctrl + F

Move to the previous command in history: Up Arrow

Move to the next command in history: Down Arrow

Cancel current command/line: Ctrl + C

Clear the screen: Ctrl + L

Exit the terminal: Ctrl + D

Search backward through command history: Ctrl + R

Redo last action (in some terminals): Ctrl + Y

Undo last action (in some terminals): Ctrl + _
